Dependencias

Metodo

Separar acervo interno da vitrine pública.

Passo 1 O site público vive em `public-site/`, sem depender dos arquivos de bastidor.
Passo 2 O workflow de Pages publica apenas esse pacote estático.
Passo 3 O repositório continua preservando o material completo para evolução do projeto.
Beneficio

Menos atrito para compartilhar.

  • Navegação mais clara para quem chega de fora.
  • Menor chance de expor arquivos internos como homepage pública.
  • Replicação simples do padrão que já funcionou na Elevra.
