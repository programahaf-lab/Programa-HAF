Documentacao comercial organizada para consulta e apresentacao.

O MEI Docs transforma um conjunto de materiais técnicos e comerciais em uma camada de navegação mais clara, leve e publicável. A base segue o mesmo método aplicado no projeto Elevra Sunwear, agora adaptado para um repositório com publicação no GitHub Pages.

Narrativa

Uma camada mais apresentável para um acervo técnico.

O site concentra visão geral, contexto de entrega e acesso ao material-fonte sem depender de ferramentas específicas ou de uma pilha frontend complexa.

Uso
  • Apresentação comercial em reuniões
  • Hub de documentação navegável
  • Ponto único para evolução pública do projeto

O que este site resolve

Em vez de expor arquivos soltos como entrada principal, o Pages passa a entregar uma experiência objetiva: visão do projeto, lógica da documentação e caminhos claros para navegação.

1

Contexto primeiro

O visitante entende rapidamente o que é o projeto antes de abrir arquivos ou materiais complementares.

2

Publicacao limpa

O deploy publica apenas o que é necessário para o site, sem depender do restante do acervo do repositório.

3

Manutencao simples

Tudo é HTML estático, fácil de ajustar, versionar e republicar.

Estrutura adotada para o GitHub Pages

O projeto agora separa o conteúdo publicado da base interna do repositório. Isso reduz atrito, evita links quebrados na vitrine pública e deixa o fluxo de deploy mais previsível.

Site publico

`public-site/`

Contém `index.html`, a página principal, o deck e `.nojekyll`, exatamente o que o Pages precisa para servir o site.

Repositorio

Acervo completo

Os materiais técnicos e comerciais continuam no repositório como fonte de trabalho, sem precisar virar a homepage pública.

Deploy

Workflow dedicado

Um workflow publica apenas a pasta pública usando GitHub Actions, no formato recomendado pela documentação oficial do GitHub Pages.

Navegacao

Deck separado

A apresentação executiva fica em uma página própria, melhor para reuniões e compartilhamento rápido.

Acesso e continuidade

A vitrine pública ficou propositalmente enxuta. O material complementar permanece disponível no próprio repositório para aprofundamento e evolução do projeto.
